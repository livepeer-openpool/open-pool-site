# Open Pool Website

THIS PROJECT is derived from [Livepool](https://github.com/Livepool-io), as such all code is open-sourced and MIT license.

This site uses NuxtJS. 

## Installation

```
npm install
```

## Run Development

```
npm run dev
```

## Build

```
npm run build
npm run generate
```

## Run Production

```
npm start
```
